# AI Usage Declaration

## How I Used AI in This Project

### Claude API — Core Feature
The main AI feature of this app is the gemstone recommendation engine. When a user fills out their details and submits the form, the backend sends their zodiac sign, birthdate, and selected life challenges to Claude AI. Claude analyzes this and returns 3 personalized gemstone recommendations in JSON format. Each recommendation includes the gem name, its properties, why it suits that particular user, and how to use it.

The Claude model used is: **claude-sonnet-4-6** (referenced in configuration as `claude-3-5-sonnet-20241022` or custom env overrides).

### AI Tools Used During Development
During the building of this app, I used AI tools in these ways:
- Claude and ChatGPT to bounce ideas, write code snippets, and debug tricky components.
- AI assistance to structure the folders, build validation logic, and sort out configuration files.
- AI helpers to write the text content for the AstroGem informational About page (the gemstone benefits, zodiac pairings, and historical descriptions).

### What AI Did NOT Do
- All final decisions on the folder setup, app flow, and features were made by me.
- The core database schemas, page routing, and frontend theme colors were designed by me.
- AI suggestions were always reviewed, cleaned up, and customized before placing them into the app.

### Important Note
The gemstone recommendations generated by this app are for spiritual and wellness purposes only. They are not medical advice.
